I’ve been doing some work with RNA velocity lately and I thought this was a good opportunity to write my first blog post. I’m going to go through different aspects of it pretty slowly over a few posts, hence the “low velocity” in the title.

Background reading

If you want to read about the biology behind RNA velocity you can go to the original publication. If that takes you to a paywall you can read the preprint or get a pdf from the first author’s website la manno lab > publications. The software associated with the publication lives at velocyto.org. Later on I’ll be using scvelo which you can read about in this preprint.

Some helpful background to modelling gene expression with differential equations can be found in:

  • Mathematical modeling of gene expression: a guide for the perplexed biologist
  • Dr Hosam Abdel Aleem’s PhD thesis
